Next, I incorporated Principal Component Analysis (PCA) as a dimensionality reduction technique. PCA allows the transformation of our high-dimensional feature space into a lower-dimensional representation while keeping "important" information. By using the principal components that explain the majority of the variance in the data, PCA enables us to reduce multicollinearity among features, just like stated up above. This helps simplifies the modeling process.¶

from sklearn.decomposition import PCA
from sklearn.svm import SVC
from sklearn.pipeline import Pipeline
data = load_breast_cancer()
X = data.data
y = data.target
pca = PCA(n_components=2)
svc = SVC()
model = Pipeline([('pca', pca), ('svc', svc)])
model.fit(X, y)
